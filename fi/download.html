OmegaT:n vanhoille käyttäjille tuttu itse asennettava versio. Jos olet epävarma, valitse tämä versio.

Web Start -versio on asenna-ja-unohda-versio.
Hyvä puoli: Kun olet kerran käynnistänyt tämän, kaikki tulevat Web Start -päivitykset asentuvat tietokoneellesi automaattisesti.
Huono puoli: Versiota ei tällä hetkellä voi käyttää yhdessä laajennusten kanssa, joten kaikki toiminnot eivät ole saatavilla.

OmegaT:n voi asentaa mihin tahansa käyttöjärjestelmään, jossa Java Runtime Environment 1.6 toimii.

Valitse, lataatko OmegaT:n ilman Java Runtime Environmentia (JRE) vai sen kanssa. Jos Windows-koneellasi on jo sopiva JRE, valitse versio ilman JRE:tä. Jos sitä ei ole tai et ole varma, valitse versio, jossa on JRE.

Tavallisesti OmegaT toimii kaikissa Linux-järjestelmissä, joihin voi asentaa sopivan Java Runtime Environment (JRE) -version. Tällä hetkellä OmegaT vaatii JRE-version 1.6 tai uudemman. Voit tarkistaa, mikä JRE Linuxissasi on tai onko sitä ollenkaan kirjoittamalla konsoliin (komentoikkunaan):

$ java -version

Jos et ole varma, onko järjestelmässäsi sopivaa JRE:tä, voit tallentaa ja asentaa OmegaT-version, jossa on JRE. Tätä JRE-versiota on turvallista käyttää, sillä se toimii ainoastaan paikallisessa OmegaT:ssä eikä aiheuta ristiriitaa muiden JRE-asennusten kanssa tai korvaa tai poista niitä.

OmegaT:n "paristollinen" versio, jossa oma Java Runtime Environment. Tämän version lataaminen kestää muita kauemmin, ja se vie enemmän tilaa kovalevyllä, mutta voit olla varma, että se toimii. Jos et ole varma järjestelmäsi sisällöstä, valitse tämä vaihtoehto.

Asennuksen mukana tulee 64-bittinen JRE.

OmegaT:n standardiversiosta on täydellinen käsikirja (ainakin englanniksi), ja sen testauksen aikana löytyneet on korjattu. Tämä on uusille OmegaT:n käyttäjille suositeltu versio.

Kuten nimestä voi päätellä, OmegaT:n uusin versio on perusversiota päivitetympi. Käsikirja voi kuitenkin olla vanhentunut (ja siksi hämmentävä käyttää), ja ohjelmassa voi olla joitakin bugeja. Uusimmat OmegaT-versiot ovat silti melko vakaita, ja niitä suositellaan kokeneemmille OmegaT-käyttäjille.

Vaatimukset

Järjestelmävaatimukset

OmegaT toimii kaikissa järjestelmissä, joihin sopiva JRE (Java Runtime Environment) joko on asennettu tai voidaan asentaa. Nykyisin OmegaT:n mukana saa sopivan JRE:n, eikä sitä tarvitse asentaa erikseen. OmegaT:n asentaminen onnistuu Windosiin (versiosta 98 alkaen), Maciin ja Linuxiin.

Microsoft Officen XML-muotoisia tiedostoja voi kääntää suoraan OmegaT:ssä. Vanhemmat MS Word-, Excel- ja Powerpoint-tiedostot täytyy ennen kääntämistä muuttaa vastaaviin nykyisiin Microsoft Office -muotoihin tai Office Open XML -muotoon, jonka voi tehdä OpenOffice.orgissa tai LibreOfficessa (sielläkin vapaasti). Vasemmalta löytyvästä linkistä Kolmannen osapuolen ohjelmistot löydät OpenOffice.orgin ja LibreOfficen ja voit asentaa ne itsellesi.

Käyttövaatimukset

OmegaT:tä ei ole erityisemmin hankalaa asentaa tai käyttää, mutta se vaatii halun lukea ja noudattaa ohjeistusta. Jos olet vastahakoinen lukemaan ohjeita ja mieluummin painelet uuden ohjelmiston jokaista painiketta niin kauan, että saat jotakin halutun kaltaista näkyviin, OmegaT ei välttämättä sovi sinulle.

Ennen kuin jatkat

Oletko miettinyt lahjoitusta OmegaT-projektille? Vaikka OmegaT on ilmainen, mutta ohjelma on mielestäsi hyödyllinen ja haluat kiittää siitä, voit tehdä lahjoituksen PayPal-tilillemme.

Vanhat versiot

OmegaT:n vanhempia ja korvattuja versioita on edelleen saatavilla.

Vanhat versiot ovat sellaisia, jotka uudemmat ja kehittyneemmät versiot ovat syrjäyttäneet. Ne eivät ole ajankohtaisia useimmille käyttäjille. Jotkut kuitenkin saattavat törmätä (joko käyttöjärjestelmän tai Java-version asettamiin) rajoituksiin, minkä vuoksi he suosivat OmegaT:n vanhempaa versiota.

JRE:llinen tai ilman sitä

Jotta OmegaT toimisi, koneellasi on oltava Java Runtime Environment (JRE).

Jos olet varma siitä, että koneellasi on jo sopiva JRE-versio, voit asentaa OmegaT:n, jossa ei ole JRE:tä. Lataaminen käy nopeasti, ja asennus vaatii vain vähän tilaa kovalevyllä. Älä tällöin valitse oletusversiota, vaan joku muu (esim. standardi tai uusin versio), ja valitse sitten versio, jossa ei ole JRE:tä.

Jos asennat OmegaT-version, jossa on JRE, vaikka koneellasi on jo JRE asennettuna, OmegaT:n JRE asennetaan erillisenä, eikä se aiheuta ristiriitaa jo asennetun version kanssa. Siksi JRE:llinen OmegaT on aina turvallinen asentaa. Jos et ole varma, onko käytössäsi sopivaa JRE:tä, valitse versio, jossa on JRE (tai oletusversio).

Linux-käyttäjille: Useimmissa Linux-jakeluissa on Java Runtime Environment, mutta joissakin avoimen lähdekoodin ympäristöissä oleva versio ei toimi kunnolla OmegaT:n kanssa (vaikka sitä ei välttämättä huomaa). Jos Linux on sinulle suhteelisen uusi tuttavuus, sinun kannattaa valita OmegaT, jossa on JRE. Kokeneemmille Linux-käyttäjäjille voi olla parempaa asentaa Oraclen (entisen Sunin) JRE järjestelmätasolle.

Allekirjoitettu/allekirjoittamaton

Gatekeeper on macOS:n ominaisuus, joka tekee allekirjoittamattomien ohjelmien ajamisesta vaikeaa ja parantaa näin turvallisuutta. OmegaT:stä on olemassa sekä allekirjoitettu että allekirjoittamaton versio Maciin. Niissä on samat toiminnot, mutta on suositeltavaa käyttää allekirjoitettua versiota, sillä järjestelmä tunnistaa sen tunnistetun kehittäjän ohjelmaksi.

Allekirjoitettujen versioiden julkaisu voi tapahtua viiveellä. Jos sinulla on kiire, voit käyttää allekirjoittamatonta versiota. Jos ohjelman käynnistäminen on vaikeaa, tarkista ohjeet tunnistamattomalta kehittäjältä tulevan ohjelman asentamiseen .

Web Start -versio tarjoaa vaihtoehtoisen tavan asentaa OmegaT koneellesi.

Tässä versiossa tulevat päivitykset asennetaan automaattisesti.

Asennuksen aikana saatat saada useita turvallisuusvaroituksia, käyttöjärjestelmäsi riippuen.
Sertifikaatin on allekirjoittanut PnS Concept Sarl, Ranskassa rekisteröity yritys.

Oikeudet, jotka annat tälle versiolle (vaikkapa "rajoittamaton tietokoneen käyttöoikeus"), ovat täsmälleen samat kuin paikallisella versiolla, eli se voi käyttää tietokoneen kovalevyä.

Koneellasi on oltava Java 1.3 tai uudempi, jotta voit asentaa Web Start -version.

Tietosuojatiedot: OmegaT Java Web Start ei tallenna mitään tietoja käyttäjästä OmegaT:n verkkopalvelimelle. Sovellus on käynnissä ainostaan omalla koneellasi. Käyttämäsi asiakirjat ja käännösmuistit pysyvät omalla koneellasi, eikä OmegaT-projektilla ole pääsyä työhösi tai tietoihisi.

Alfavaiheen master-versio edustaa kehityksen viimeisintä vaihetta, Se on tarkoitettu ainoastaan todellisille kehittäjille ja testaajille. Version saa myös git-komennolla

git clone https://git.code.sf.net/p/omegat/code omegat-code

Öiset käännökset löytyvät Sourceforgesta.

Tavallinen käyttäjä ei tarvitse lähdekoodipaketteja, vaan ne on tarkoitettu vain ohjelmistokehittäjille. Paketit löytyvät vastaavien käännettyjen versioiden kansioista.

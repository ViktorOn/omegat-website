用户手册

OmegaT 附带有全面详细的用户手册。要从 OmegaT 中调出手册，只需选择“帮助 > 用户手册”即可，或按 F1。

默认情况下，OmegaT 用户界面上会显示带你全面了解 OmegaT 主要操作和功能的《快速入门指南》。

你也可以在不启动 OmegaT 本身的情况下调出附带的用户手册。可以在 OmegaT 程序文件夹中的 docs 文件夹内找到用户手册。

在线用户手册 PDF 格式的 OmegaT 用户手册（OmegaT 4.3.3）。
在线 HTML 用户手册 HTML 格式的 OmegaT 标准版用户手册。

OmegaT pote ser installate sur tote le systemas operative capabile de facer fluer le version 1.6 de Java Runtime Environment (JRE).

Selige si discargar OmegaT sin o includente Java Runtime Environment (JRE). Si sur tu systema Windows es jam installate un convenibile version del JRE, selige "sin JRE". Si non, o si tu non es secur, selige le version "Con JRE".

OmegaT essera habitualmente exequite sur omne le systema Linux sur le qual pote esser installate un convenibile version del Java Runtime Environment (JRE). OmegaT actualmente require le version 1.6 de JRE o plus tarde. Pro verificar si alicun version del JRE es installate sur tu systema Linux, aperi un consola (fenestra pro linea de commando) e scribe:

$ java -version

Si tu non es secur si un convenibile JRE es jam installate sur tu systema, tu pote totevia discargar e installar un version de OmegaT con JRE. Iste version del JRE essera usate sol localmente pro OmegaT, illo non confligera, non habera plus importantia e non disinstallara ulle altere installation de JRE, e dunque illo es secur a usar.

Le version de OmegaT "batterias includite": con su proprie JRE includite. Iste version prendera plus de tempore pro discargar e requirera plus de spatio sur tu disco dur, ma tu pote fider que illo fluera. Si tu es dubitose, selige iste option.

Le version "Standard" de OmegaT include un complete manual del usator (al minus in anglese) e habitualmente essere probate pro alicun tempore pro eliminar alicun defectos. Illo es le version recommendate pro le neophytos.

Como su nomine suggere, le "Ultime" version de Omegat es le plus actualisate. Comocunque, le manual del usator pote esser non jam actualisate (e dunque possibilemente confuse), e qualque minor defectos es possibile. Totevia, le versiones "Ultime" de OmegaT es habitualmente multo stabile e es recommendate pro omne usatores qui jam ha alicun experientia con OmegaT.

Requisitos

Requisitos del systema

OmegaT fluera sur tote le systemas operative sur le qual on pote installar un convenibile version del Java Runtime Environment (JRE). Le JRE es anque supplite con OmegaT, on non debe lo obtener separatemente. OmegaT esseva installate con bon successo sur Windows (tote le versiones a partir de Windows 98 in avante), Macintosh OS X e Linux.

On pote render, directemente in OmegaT, le files in formato Microsoft Office XML. On pote alsi render le files in senior formatos MS Word, Excel e Powerpoint, con le previe conversion al actual formatos equivalente de Microsoft Office, o al formato Office Open XML, como on pote facer in OpenOffice.org o LibreOffice (alsi gratuite). Pro obtener OpenOffice.org o LibreOffice, seque le ligamine ex le button "Software de tertie partes" a sinistra.

Requisitos del usator

OmegaT non es particularmente difficile a installar o usar, ma illo require bon voluntate pro leger e sequer le instructiones. Si tu ha un natural aversion al lectura del instructiones, e tu approche a nove software es de cliccar sur cata button tu vide usque alicun cosa resimilante al effecto desiderate occurre, alora OmegaT non es probabilemente convenibile pro te.

Version standard - OmegaT 4.3.2

Le version Standard esseva habitualmente in uso pro alicun tempore e es dunque experimentate. Su documentation, al minus in anglese, es de costume rationabilemente actualisate. Illo tamen mancara de qualque characteristicas del version "Ultime".

Versiones Vetule

Tu pote ancora trovar versiones plus vetere, jam supplantate, de OmegaT.

Le versiones vetule es senior versiones que esseva supplantate. Pro le major parte del usatores, istos non es interessante. Comocunque, alicun usatores pote affrontar restrictiones (imponite ab lor systemas operative o version de Java) que pote resultar in un preferentia pro un version plus vetule.

Con/sin JRE

Pro fluer, OmegaT besonia que le Java Runtime Environment (JRE) es installate sur tu computator.

Si tu es confidente que sur tu computator es jam installate un convenibile version del JRE, tu pote discargar un version sin le JRE de OmegaT. Le discarga essera plus rapide, e le installation requirera minus spatio sur tu disco dur. In iste caso, non seliger le version Default; selige un version differente (e.g. Standard o Ultime), deinde selige un version marcate "Java not included".

Si tu installa le version de OmegaT con JRE, anque si le JRE es ja installate sur tu computator, le JRE de OmegaT essera installate separatemente e non confligera con illo. Il es ergo sempre secur installar le version de OmegaT con JRE. Si tu non es secur de haber un version convenibile del JRE, selige un version con JRE (o le version Default).

Usatores de Mac OS X: JRE es ja installate normalmente sur Mac OS X; in consequentia non ha ulle version con JRE pro iste systema operative.

Usatores de Linux: le major parte del distributiones de Linux es supplite con JRE, ma ulle installationes open-source non functiona debitemente con OmegaT (ben que isto pote non parer subito). Si tu es relativemente nove re Linux, nos te consilia de seliger un version de OmegaT con le JRE includite. Le usatores plus experte de Linux pote preferer de installar le JRE de Oracle, olim Sun, a nivello de systema.

Signate/Non signate

Mac OS X ha un functionalitate nominate Gatekeeper que accresce le securitate per render difficile facer fluer programmas non signate. OmegaT pro Mac OS X es disponibile in ambe le versiones, signate e non signate; illos es functionalmente identic, ma le version signate es ideal pois que illos es recognoscite per le systema como esser publicate per un note disveloppator.

On pote haber un retardo a publicar le versiones signate. Le usatores hastive pote usar le version non signate in vice; si tu ha difficultate a lancear le app, vide como aperir un app de un disveloppator non identificate.

Le version Web Start offere un medio alternative pro installar OmegaT sur tu computator.

Con iste version, le futur nove versiones essera installate automaticamente.

Durante le installation, dependente de tu systema operative, tu pote reciper plure advertimentos de securitate.
Le certificato es signate per PnS Concept Sarl, un compania registrate francese.

Le permissiones que tu da a iste version (le qual pote esser describite como "accesso sin restriction al computator") non es differente del permissiones que tu assigna al version local, id es un accesso al disco dur de tu computator.

Pro installar le version Web start, tu debe jam haber installate sur tu computer le version Java 1.3 o plus tarde.

Information re le protection del confidentialitate: OmegaT Java Web Start registra nihil re tu informationes in le servitores de OmegaT. Le application flue sur tu machina. Tu documentos e tu memorias de traduction remane sur tu computator, e le projecto OmegaT habera nulle accesso a tu labor o a tu informationes.

Le version /trunk (alpha) reflecte le ultimissime stato del disveloppamento. Illo es recommendate solmente pro disveloppatores experte e probatores. On pote anque lo obtener per le commando subversion:

svn checkout svn://svn.code.sf.net/p/omegat/svn/trunk omegat-svn

Compilationes periodic es disponibile sur Sourceforge.

Le pacchettos de codice fonte non es requirite per le usatores e es designate sol pro le disveloppatores. On los pote trovar in le plicas del versiones compilate correspondente.

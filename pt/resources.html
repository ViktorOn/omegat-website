Utilities

Macros de segmentaço para OpenOffice.org Contribuiço de Benjamin Siband. Permite segmentar o texto por frase em vez de ser por parágrafo.
Leia-me (ReadMe) Descritivo dos macros de segmentaço feitos por Benjamin Siband
stripIT macro Macro para remover etiquetas de ficheiros TMX. Contribuiço de Sonja Tomaskovic.
Utilitário de alinhamento Contribuiço de Dmitri Gabinski. Escrito em Python.
TMXMerger Java command-line script for merging two or more TMX files. Contributed by Henry Pijffers.
TMXCleaner Java command-line script for removing "superfluous" translation units from a TMX file: deletes all TUs without target segment or in which the target segment is identical to the source segment. Contributed by Henry Pijffers.
Extspell Extspell verificador de ortografia externo. Contribuiço de Marc Prior. Necessita de tcl/tk e Aspell ( ambos Software Livre em todas as plataformas).
Conversor de ficheiro TMX do Wordfast Converte memórias de traduço em TMX (Wordfast) num formato aceite pelo OmegaT. Contribuiço de Dmitri Gabinski.

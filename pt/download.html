Requisitos

Requisitos do sistema

O OmegaT funcionará em qualquer sistema em que o JRE (Java Runtime Environment) apropriado tenha sido ou possa ser instalado. O JRE é agora fornecido com o OmegaT e não precisa de ser obtido separadamente. OmegaT foi instalado com sucesso no Windows (todas as versões a partir da 98), Macintosh OS X e Linux.

Os ficheiros em formato Microsoft Office XML podem ser traduzidos directamente no OmegaT. Os ficheiros nos formatos antigos MS Word, Excel e Powerpoint podem ser traduzidos após conversão para os formatos equivalentes actuais do Microsoft Office, ou alternativamente para o formato Office Open XML, que pode ser feito com o OpenOffice.org ou o LibreOffice (também gratuito). Para obter o OpenOffice.org ou o LibreOffice, siga a ligação a partir do botão "Programas de terceiros" à esquerda.

Requisitos do utilizador

O OmegaT não é particularmente difícil de instalar ou utilizar, mas requer alguma vontade de ler e seguir as instruções. Se tem uma aversão natural à leitura de instruções e a sua abordagem a um novo programa é clicar em cada botão que vê até que algo semelhante ao efeito desejado ocorra, então o OmegaT provavelmente não é adequado para si.

Versões antigas

Ainda pode encontrar versões antigas já substituídas do OmegaT.

As versões legadas são versões mais antigas que foram substituídas. Para a maioria dos utilizadores, estas não são de interesse. No entanto, alguns utilizadores podem enfrentar restrições (impostas pelos seus sistemas operativos ou versão Java) que podem resultar numa preferência por uma versão mais antiga.

Com/Sem JRE

Para funcionar, o OmegaT requer que o Java Runtime Environment (JRE) esteja instalado no seu computador.

Se estiver confiante de que uma versão adequada do JRE já está instalada no seu computador, pode transferir uma versão do OmegaT sem o JRE. A transferência será mais rápida e a instalação exigirá menos espaço no seu disco rígido. Neste caso, não seleccione a versão pré-definida; seleccione uma versão diferente (por exemplo, Standard ou Latest), depois seleccione uma versão marcada com "Java não incluído".

Se instalar a versão OmegaT com o JRE mesmo que o JRE já esteja instalado no seu computador, o JRE do OmegaT será instalado separadamente e não criará conflitos. Por conseguinte, é sempre seguro instalar a versão do OmegaT com JRE. Se não tiver a certeza se tem ou não uma versão adequada do JRE, seleccione uma versão com o JRE (ou a versão pré-definida).

Utilizadores Linux: a maioria das distribuições Linux são fornecidas com um Java Runtime Environment, mas algumas implementações open-source não funcionam correctamente com o OmegaT (embora isto possa não ser imediatamente visível). Se é relativamente novo no Linux, aconselhamo-lo a seleccionar uma versão OmegaT com o JRE incluído. Os utilizadores mais experientes de Linux podem preferir instalar o Oracle (anteriormente Sun) JRE a nível do sistema.

Autenticada vs Não assinada

O macOS tenta aumentar a segurança, dificultando a execução de programas não verificados.

O OmegaT para macOS está disponível nas versões Autenticada e Não Assinada; são funcionalmente idênticas, mas a versão Autenticada é preferida porque é reconhecida pelo sistema como sendo a) publicada por um programador conhecido, e b) analisada pela Apple.

Pode haver um atraso na publicação da versão autenticada. Os utilizadores com pressa podem utilizar a versão não assinada; se tiver dificuldade em iniciar a aplicação, veja como abrir uma aplicação de um programador não identificado.

A versão Web Start oferece um meio alternativo de instalar o OmegaT no seu computador.

Com esta versão, as futuras actualizações serão instaladas automaticamente.

Durante a instalação, dependendo do seu sistema operativo, poderá receber vários avisos de segurança.
O certificado é assinado pelo programador principal do OmegaT.

As permissões que dá a esta versão (que podem ser descritas como "acesso sem restrições ao computador") não são diferentes das permissões que atribui à versão local, ou seja, um acesso ao disco rígido do seu computador.

Para instalar a versão Web Start, já deve ter a versão Java 1.8 ou posterior instalada no seu computador.

Informação sobre privacidade: o OmegaT Java Web Start não guarda nenhuma das suas informações nos servidores web do OmegaT. A aplicação é executada na sua máquina. Os seus documentos e memórias de tradução permanecem no seu computador e o projecto OmegaT não terá acesso ao seu trabalho ou informação.

A versão "Master" (alfa) reflecte o mais recente estado de desenvolvimento. É recomendada apenas para programadores e equipas de teste. Também pode ser obtida usando o comando git:

git clone https://git.code.sf.net/p/omegat/code omegat-code

Estão disponíveis periodicamente novas versões na SourceForge.

Os pacotes de código-fonte não são necessários aos utilizadores e destinam-se apenas a programadores. Podem ser encontrados nas pastas das versões compiladas correspondentes.

Per verificà chì versione hè presente nant’à u vostru sistema Linux, aprite una cunsola (finestra di linea di cumanda) è entrite què :

Stu metoda hè più sicuru è impedisce tuttu cunflittu di versioni (risicu di rimpiazzamentu o disinstallazione) perchè stu JRE serà solu impiegatu lucalmente da OmegaT.

Cundizioni richieste

Cunfigurazione sistema

OmegaT pò esse lanciatu nant’à tuttu sistema chì cuntene l’ambiente d’esecuzione Java (JRE) appostu. Oghje chì u JRE hè pruvistu cù OmegaT, ùn hè più bisognu di scaricalu staccatamente. L’installazione hè stata riisciuta nant’à tutte e piattaforme Windows (da a versione 98 fin’à oghje), OS X Macintosh è Linux.

I schedarii Microsoft Office à u furmatu XML ponu esse tradutti direttamente cù OmegaT. Invvece chì quelli in un furmatu più vechju di MS Word, Excel o Powerpoint devenu esse cunvertiti, in primu locu, in i furmati equivalenti d’OpenOffice.org, d’Apache OpenOffice o di LibreOffice (seguiti di prugrammi di scagnu, libere è di rigalu). Pudete ottene OpenOffice.org o LibreOffice seguitendu a leia pruvista in a parte « Prugrammi terzi » di a pagina « Risorse ».

Prufilu utilizatore

Bench’è l’installazione è l’adopru d’OmegaT ùn fussinu micca difficiule, ellu hè assai ricumandatu di leghje a documentazione distribuita è di seguità l’istruzzioni pruviste. S’è, da bona regula, ùn lighjite micca l’istruzzioni, è chì vi scuprite un novu prugramma clicchendu dapertuttu, è aspettendu ch’accade qualcosa chì s’assumiglia à l’effettu ricercatu, tandu OmegaT ùn hè micca fattu per voi.

Versioni anziane

Pudete sempre scaricà d’anziane versioni d’OmegaT.

E versioni anziane sò vechje versioni chì sò state rimpiazzate. Per a maiò parte di a ghjente, anu pocu interessu. Sarrimanenti, in certi casi, ste versioni anziane ponu esse ghjuvevule quandu ci sò ristrizzioni imposte da u sistema d’operazione o da a versione Java.

Cù o senza JRE

Per pudè funziunà, OmegaT richiede chì l’ambiente d’esecuzione Java (JRE) sia installatu nant’à u vostru urdinatore.

S’è vo site sicuru(a) chì una versione apprupriata di l’ambiente JRE hè dighjà installata nant’à u vostru urdinatore, pudete scaricà una versione d’OmegaT senza JRE. U so scaricamentu serà più rapidu è a so installazione richiiderà menù di spaziu nant’à u vostru discu duru. In stu casu, ùn selezziunate micca a versione predefinita ma preferite piuttostu una versione « Classica » o « Ultima » eppò selezziunate una versione cù a menzione « Java not included » o « without JRE ».

S'è vo devite installà a versione d’OmegaT cù l’ambiente JRE quandu u JRE hè dighjà installatu, ùn ci seria micca cunflittu perchè e duie installazioni sò distinte. Dunque, l’installazione d’OmegaT cù u JRE hè a metoda a più sicura. S’è ùn site micca sicuru(a) d’avè una versione apprupriata di u JRE, selezziunate a versione d’OmegaT cù JRE (o a versione predefinita).

Nota per l’utilizatori Linux : a maiò parte di e distribuzioni Linux sò pruviste cù un ambiente d’esecuzione Java ma certe messe in opera « Open Source » ùn funziuneghjanu micca currettamente cù OmegaT, penseru chì ùn si vede micca subitu subitu. Ricumandemu à i principianti Linux di sceglie una versione d’OmegaT cù JRE. L’utilizatori più tercani puderanu preferisce una installazione di u JRE d’Oracle (Sun tempi fà) à u livellu di u sistema.

Firmatu/Senza firma

Mac OS X pussede una funzione chjamata Gatekeeper chì aumenteghja a sicurità di manera à rende più difficiule di lancià prugrammi micca firmati. OmegaT per Mac OS X hè dispunibule in e duie versioni - firmata è senza firma - tremenduie cù e listesse funzioni, ma a versione firmata hè preferita perchè hè ricunnisciuta da u sistema cum’è publicata da un sviluppatore cunnisciutu.

A versione « Web Start » hè una suluzione d’installazione simplice permettendu d’impiegà a versione attuale d’OmegaT.

Cù sta versione i rinnovi dispunibule sò installati autumaticamente.

Durante l’installazione, secondu à u vostru sistema d’operazione, pudete riceve parechji avertimenti di sicurità.
U certificatu hè firmatu da PnS Concept Sarl, una cumpagnia francese.

I permessi chì vo cuncidite à sta versione (chì ponu esse discritti cum’è un accessu « senza limita à l’urdinatore ») sò listessi chì quelli chì vo cuncidite à a versione lucale (i.e. : un accessu à u discu duru di l’urdinatore).

L’installazione di a versione « Web Start » richiede chì a versione 1.3 (o ancu più recente) di Java sia installata nant’à u vostru urdinatore.

Impegnu di cunfidenzialità : a versione « Web Start » d’OmegaT ùn arregistra alcunu di i vostri dati nant’à i servitori d’OmegaT. L’appiecazione funziuneghja nant’à a vostra mascina. I vostri ducumenti è e vostre memorie di traduzzione stanu nant’à u vostru urdinatore è u prughjettu OmegaT ùn averà alcunu accessu à i vostri travaglii o infurmazioni.

A versione « alfa » cuntene a fasa a più recente di u sviluppu di l’appiecazione OmegaT. Hè ricumandata solu per i sviluppatori è esperimentadori tercani. Pò dinù esse ottenuta cù quella cumanda git :

git clone https://git.code.sf.net/p/omegat/code omegat-code

Custruzzioni periodiche sò dispunibule nant’ Sourceforge.

I pacchetti cuntenendu u codice di fonte ùn sò micca richiesti per l'adopru di u prugramma è sò fatti solu per i sviluppatori. Sò piazzati in i cartulari di e versioni custruite chì currispondenu.

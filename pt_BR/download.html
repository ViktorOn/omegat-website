Requisitos

Requisitos do sistema

O OmegaT roda em qualquer sistema em que o JRE (Java Runtime Environment) possa ser ou já esteja instalado. O JRE atualmente pode ser adquirido com OmegaT e não precisa ser baixado separadamente. O OmegaT pode ser instalado em Windows (em todas as versões a partir da 98), em mac OS X e em Linux.

Arquivos no formato XML do Microsoft Office podem ser traduzidos diretamente no OmegaT. Arquivos em formatos mais antigos do MS Word, Excel e Powerpoint podem ser traduzidos desde que sejam convertidos para formatos equivalentes atuais do Microsoft Office, ou, como alternativa, para formato Office Open XML. Neste caso, a conversão pode ser feita no website OpenOffice.org ou no programa LibreOffice (que também é grátis). Para obter o OpenOffice.org ou o LibreOffice, siga o link em nosso botão de "Software de terceiros" que fica à esquerda.

Requisitos do usuário

O OmegaT não é particularmente difícil de instalar ou usar, mas requer disposição do usuário para ler e seguir as instruções. Caso tenha uma aversão natural a instruções em forma escrita e sua abordagem a novos programas seja clicar em todos os botões até que algo com mais ou menos o efeito desejado ocorra, o OmegaT, então, provavelmente não é para você.

Versão estável - OmegaT 4.3.2

A versão estável é a que normalmente já está sendo utilizada há algum tempo e, por isso, já foi aprovada. O manual dessa versão, ao menos em inglês, costuma ser bem atualizado. Essa versão não possui, porém, alguns dos recursos da "última versão".

Versão legacy

É possível encontrar versões mais antigas e ultrapassadas do OmegaT.

As versões legacy são versões mais antigas que estão obsoletas. Para a maioria dos usuários, essas versões não são interessantes. Porém, alguns usuários podem encontrar restrições (impostas pelo sistema operacional ou pela versão do plugin Java deles) que pode fazer com que prefiram uma versão mais antiga.

Com ou sem JRE

O OmegaT precisa do Java Runtime Environment (JRE) instalado em seu computador para rodar.

Caso tenha certeza de que uma versão adequada do JRE já esteja instalada em seu computador, pode baixar uma versão do OmegaT sem o JRE. Ela leva menos tempo para ser baixada e ocupa menos espaço em seu HD. Se esse for o caso, não escolha a versão "padrão": escolha uma versão diferente (a "última versão" ou a "versão estável", por exemplo) e então escolha uma versão que tenha a opção "Java não incluso" marcada.

Caso instale a versão do OmegaT com o JRE incluso, mesmo que já tenha instalado o JRE em seu computador, essa versão será instalada separadamente e não entrará em conflito com a outra. Portanto, é sempre seguro instalar a versão do OmegaT com JRE. Caso não tenha certeza se seu computador já possui uma versão adequada do JRE, escolha uma versão com JRE incluso (ou a versão padrão).

Usuários de Mac OS X: o JRE já vem instalado no Mac OS X por padrão, portanto, não há versões para Mac OS X com o JRE incluso.

Usuários de Linux: a maioria das versões do Linux já tem o Java Runtime Environment, mas algumas implementações em código aberto não funcionam corretamente com o OmegaT (embora isso possa não ser aparente de imediato). Caso seja um usuário novo de Linux, aconselhamos escolher uma versão com o JRE incluso. Usuários mais experientes podem preferir instalar o JRE da Oracle (antiga Sun) em nível de sistema.

Programas assinados/não assinados

O Mac OS X possui um recurso chamado Gatekeeper, que aumenta a segurança do sistema e dificulta a execução de programas não assinados. O OmegaT para Mac OS X está disponível em versões assinada e não assinada. Ambas têm a mesma funcionalidade, mas a versão assinada é a mais escolhida pelos usuários, já que é reconhecida pelo sistema como uma versão publicada por uma desenvolvedora conhecida.

Pode haver um atraso na publicação de versões assinadas. Por isso, usuários com pressa podem preferir utilizar a versão não assinada. Caso tenha dificuldades ao iniciar o aplicativo, veja como abrir o aplicativo de uma desenvolvedora não identificada..

A versão Web Start oferece formas alternativas de instalar o OmegaT em seu computador.

Com essa versão, as atualizações futuras serão instaladas automaticamente.

Durante a instalação, dependendo de seu sistema operacional, você pode receber diversos avisos de segurança.
O certificado é assinado pela PnS Concept Sarl, uma empresa francesa registrada.

As permissões concedidas para esta versão (que podem ser descritas como "acesso irrestrito ao computador") não são diferentes das permissões concedidas à versão local, como conceder acesso ao HD de seu computador.

Para instalar a versão Web start, a versão 1.3 ou superior do Java precisa estar instalada em seu computador.

Informação de privacidade: o Java Web Start do OmegaT não salva nenhuma de suas informações nos servidores do OmegaT. O aplicativo é executado na sua máquina. Os documentos e memórias de tradução permanecem em seu computador, e o OmegaT não tem nenhum acesso ao seu trabalho ou às suas informações.

A versão /trunk (alpha) representa o último estado em desenvolvimento da ferramenta. Essa versão é recomendada apenas para desenvolvedores e analistas de teste. Ela também pode ser obtida ao usar o comando subversion:

svn checkout svn://svn.code.sf.net/p/omegat/svn/trunk omegat-svn

Builds são disponibilizadas periodicamente no Sourceforge.

Os pacotes de código-fonte não são solicitados por usuários e são direcionados somente aos desenvolvedores. Esses pacotes podem ser encontrados nas pastas das versões compiladas correspondentes.

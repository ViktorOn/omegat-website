Requisitos

Requisitos del sistema

OmegaT funcionará en cualquier sistema en el que esté instalado o pueda instalarse JRE (Java Runtime Environment). Ahora, JRE se incluye con OmegaT y no es necesario obtenerlo por separado. OmegaT se instala correctamente en Windows (todas las versiones posteriores a 98), Macintosh OS X y Linux.

Los archivos en formato XML de Microsoft Office se pueden traducir directamente en OmegaT. Los archivos en formatos antiguos de MS Word, Excel y PowerPoint se pueden traducir después de convertirlos a la versión actual de los formatos de Microsoft Office, o al formato Office Open XML, que se puede crear en OpenOffice.org o LibreOffice (también gratuito). Para obtener OpenOffice.org o LibreOffice, haga clic en el enlace del botón "Software de terceros" a la izquierda.

Requisitos del usuario

OmegaT no es especialmente difícil de instalar ni usar, pero es necesario tener la voluntad de leer y seguir las instrucciones. Si tiene una adversión natural a leer instrucciones y su forma de usar un software nuevo es hacer clic en cada botón que vea hasta que ocurra algo parecido al efecto deseado, entonces es probable que OmegaT no sea adecuado para usted.

Versiones heredadas

Aún se pueden encontrar versiones más antiguas que se han sustituido de OmegaT.

Las versiones heredadas son versiones más antiguas que se han sustituido. Para la mayoría de usuarios, carecen de interés. Sin embargo, algunos usuarios pueden encontrar restricciones (impuestas por sus sistemas operativos o versión de Java) que pueden hacer que prefieran una versión más antigua.

Con/sin JRE

Es necesario que tenga Java Runtime Environment (JRE) instalado en su equipo para que OmegaT funcione.

Si está seguro de que ya tiene una versión adecuada de JRE instalada en su equipo, puede descargar una versión de OmegaT sin JRE. La descarga será más rápida y la instalación necesitará menos espacio en su disco duro. En ese caso, no seleccione la versión predeterminada; seleccione una versión diferente (p. ej., la estándar o la última) y seleccione después una versión que no incluya Java.

Si instala la versión de OmegaT con JRE aunque ya tenga JRE instalado en su equipo, el JRE de OmegaT se instalará por separado y no entrará en conflicto. Por lo tanto, siempre es seguro instalar la versión de OmegaT con JRE. Si no está seguro de si tiene o no una versión adecuada de JRE, seleccione una versión con JRE (o la versión predeterminada).

Usuarios de Linux: la mayoría de distribuciones de Linux cuentan con Java Runtime Environment, pero algunas implementaciones de código abierto no funcionan bien con OmegaT (aunque no lo parezca en un primer momento). Si es relativamente nuevo en Linux, le recomendamos que seleccione una versión de OmegaT con JRE incluido. Los usuarios de Linux más experimentados pueden preferir instalar JRE de Oracle (antes Sun) a nivel de sistema.

Firmado/sin firmar

Mac OS X tiene una función llamada Gatekeeper que incrementa la seguridad dificultando la ejecución de programas sin firmar. OmegaT para Mac OS X está disponible en versión firmada y sin firmar; cuentan con las mismas funciones, pero es preferible usar la versión firmada porque el sistema reconoce que la ha publicado un desarrollador conocido.

Es posible que se tarde un tiempo en publicar las versiones firmadas. Los usuarios que tengan prisa pueden usar en su lugar la versión sin firmar; si tiene dificultades para ejecutar la aplicación, consulte cómo Abrir una app de un desarrollador no identificado.

La versión Web Start le ofrece una forma alternativa de instalar OmegaT en su equipo.

Con esta versión, las futuras actualizaciones se instalarán automáticamente.

Durante la instalación, dependiendo de su sistema operativo, puede recibir varias advertencias de seguridad.
El certificado está firmado por PnS Concept Sarl, una organización registrada de Francia.

Los permisos que le dé a esta versión (que se pueden describir como "acceso no restringido al equipo") son idénticos a los permisos que asigne a la versión local, es decir, permiten el acceso al disco duro de su equipo.

Para instalar la versión Web Start, es necesario que tenga la versión de Java 1.3 o posterior instalada en su equipo.

Información sobre la privacidad: OmegaT Java Web Start no guarda ninguna información en los servidores web de OmegaT. La aplicación se ejecuta en su equipo. Sus documentos y memorias de traducción permanecen en su equipo y el proyecto de OmegaT no tendrá acceso a su trabajo ni a su información.

La versión "master" (alfa) refleja el último estado de desarrollo. Se recomienda solo para desarrolladores y testers serios. También se puede conseguir usando el siguiente comando:

git clone https://git.code.sf.net/p/omegat/code omegat-code

Las compilaciones periódicas están disponibles en Sourceforge.

Los paquetes de código fuente no son necesarios para los usuarios y están pensados solo para desarrolladores. Se pueden encontrar en las carpetas de las versiones compiladas correspondientes.

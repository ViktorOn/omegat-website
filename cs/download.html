Pro instalaci na ostatních operačních systémech (jako je FreeBSD a Solaris).
OmegaT lze instalovat na různé operační systémy, které jsou schopny zprovoznit prostředí Java runtime environment ve verzi 1.6

Vyberte, zda stáhnout OmegaT s nebo bez obsaženého prostředí Java Runtime Environment (JRE). Pokud máte vhodné JRE již instalováno ve svém systému Windows, vyberte „Bez JRE“. Pokud ne, nebo si nejste jisti, vyberte „S JRE“.

OmegaT zpravidla běží na jakémkoliv Linuxu, na kterém lze instalovat vhodnou verzi prostředí Java Runtime Environment (JRE). OmegaT v současnosti vyžaduje verzi JRE 1.6 nebo novější. Pro kontrolu, kterou verzi JRE, jestli vůbec nějakou, máte instalovánu na svém Linuxu, otevřete konzoli (okno s příkazovým řádkem) a napište:

$ java -version

Pokud si nejste jistí, zda je na vašem systému již instalována vhodná verze JRE, pak si můžete stáhnout a instalovat verzi OmegaT s JRE. Tato verze JRE bude použita pouze lokálně pro OmegaT, nebude kolidovat, přepisovat nebo odinstalovávat jiné instalace JRE, a je to tedy bezpečné ji použít.

Verze OmegaT s vlastním prostředím Java Runtime Environment, u které „jsou baterie součástí balení“. Tato verze se bude déle stahovat a potřebuje více místa na vašem pevném disku, ale můžete si být jisti, že půjde spustit. Pokud si nejste jistí s výběrem verze, berte tuto.

Obsažená verze JRE je verze 64-bitová

Verze OmegaT s vlastním prostředím Java Runtime Environment, u které „jsou baterie součástí balení“. Tato verze se bude déle stahovat a potřebuje více místa na vašem pevném disku, ale můžete si být jisti, že půjde spustit. Pokud si nejste jistí s výběrem verze, berte tuto.

Pokud si nejste jisti, jestli je vhodné prostředí Java Runtime Environment již instalováno na vašem systému na systémové úrovni, můžete vybrat možnost bez JRE.

„Standardní“ verze OmegaT obsahuje kompletní uživatelskou příručku (minimálně anglicky) a zpravidla byla již nějaký čas testována, aby se vychytaly chyby. Toto je doporučená verze pro začátečníky.

Jak už napovídá název, „Nejnovější“ verze OmegaT je přeci jen aktuálnější. Nicméně, uživatelská příručka již aktuální být nemusí (a proto může být zavádějící), je možné, že v této verzi se budou vyskytovat nějaké menší chyby) Nicméně „Nejnovější“ verze OmegaT jsou zpravidla velmi stabilní a jsou doporučeny pro všechny uživatele, kteří mají již nějakou zkušenost s OmegaT.

Požadavky

Systémové požadavky

OmegaT je možno spustit pod jakýmkoliv systémem, na kterém je nainstalované nebo je možné nainstalovat prostředí JRE (Java Runtime Environment). Prostředí JRE se v současnosti dodává přímo s aplikací OmegaT a není zapotřebí si jej obstarávat zvlášť. Program OmegaT byl úspěšně instalován na systémech Windows (všechny verze od 98), Macintosh OS X a Linux.

Soubory ve formátu Microsoft Office 2007 XML je možné překládat v OmegaT přímo. Soubory ze staršího formátu MS Wordu, Excelu a PowerPointu lze překládat až po konverzi na ekvivalentní aktuální formáty Microsoft Office, nebo podobně na formát Office Open XML, což lze provést pomocí OpenOffice.org nebo LibreOffice (tento formát je také svobodný). OpenOffice.org nebo LibreOffice můžete stahovat přes link na tlačítku vlevo „Software třetích stran“.

Požadavky na uživatele

Aplikaci OmegaT není vůbec těžké nainstalovat nebo používat, ale vyžaduje to vůli číst a řídit se pokyny. Pokud máte přirozenou nevůli ke čtení pokynů, a Váš přístup k novému software je kliknout na každé tlačítko, které vidíte dokud se neobjeví něco podobající se požadovanému efektu, potom pro Vás aplikace OmegaT pravděpodobně není vhodná.

Standardní Verze - OmegaT 4.3.2

Standardní verze je obvykle již nějakou dobu používána a je tedy vyzkoušena. Její dokumentace, minimálně ta anglická, je zpravidla co nejaktuálnější. Nicméně zde budou chybět některé funkce „Nejnovější“ verze.

Zastaralé verze

Stále můžete najít starší, nahrazené verze aplikace OmegaT.

Zastaralé verze jsou starší verze, které byly již nahrazeny. Pro většinu uživatelů již nejsou atraktivní. Nicméně, někteří uživatelé musí čelit omezením (dané jejich operačním systémem či verzí Javy), které mohou mít za následek upřednostnění starší verze.

S / Bez JRE

Ke spuštění vyžaduje OmegaT ve vašem systému nainstalované prostředí Java Runtime Environment (JRE).

Pokud si jste jistí a víte, že je už na Vašem počítači nainstalované vhodné JRE, prosím stáhněte si verzi bez JRE. Stahování proběhne rychleji a instalace tak bude vyžadovat méně místa na vašem pevném disku. V tomto případě nevybírejte Výchozí verzi; vyberte jinou verzi (např. Standardní nebo Nejnovější), pak si vyberte verzi „Bez Javy“.

Nicméně, když instalujete verzi OmegaT i když je JRE na vašem počítači již instalováno, tak bude JRE pro OmegaT instalováno odděleně a nebude se stávajícím prostředím kolidovat. Proto je vždy jistější instalovat verzi OmegaT i s JRE. Pokud si nejste jisti, zda máte či nemáte odpovídající verzi JRE, vyberte si verzi s JRE (tj. výchozí verze)

Uživatelé Mac OS X: ve výchozím nastavení systému je JRE již instalováno, proto není k dispozici verze s JRE pro Mac OS X.

Uživatelé Linuxu: většina linuxových distribucí je doplněna prostředím Java Runtime Environment, ale některé implementace open-source nespolupracují v plné míře s aplikací OmegaT (i když to nemusí být zcela zjevné na první pohled). Pokud jste poměrně noví v Linuxu, doporučujeme vám vybrat si verzi OmegaT, kde je JRE obsaženo. Více zkušení uživatelé Linuxu možná budou preferovat instalaci Oracle (dříve Sun) JRE do svého systému.

Podepsané/Nepodepsané

Mac OS X má funkcionalitu zvanou Gatekeeper, která zvyšuje bezpečnost při stížení spouštění nepodepsaných programů. OmegaT pro Mac OS X je k dispozici v obou verzích, jak podepsaná, tak nepodepsaná; mají identické funkce, ale podepsaná verze je přednostně rozpoznána systémem jakožto vydaná známým vývojářem.

Při vydávání podepsaných verzí může docházet ke zpožděním. Uživatelé, kteří nechtějí čekat mohou nepodepsanou verzi také použít; pokud máte potíře se spouštěním aplikace, hledejte v nápovědě jak otevřít aplikaci od neznámého vývojáře.

Verze Web Start nabízí alternativní prostředek instalace OmegaT na váš počítač.

Prostřednictvím této verze se zajistí, že se budoucí nové verze budou instalovat automaticky.

V závislosti na operačním systému můžete obdržet několik bezpečnostních hlášení.
Certifikát podepsala společnost PnS Concept Sarl, společnost registrovaná ve Francii.

Práva, která udělíte této verzi (která lze nazývat jako „neomezený přístup k počítači“) jsou identická s právy, která jste přidělili lokální verzi při její instalaci, tj. přístup na hard disk vašeho počítače.

Aby bylo možno instalovat verzi Web Start, musíte mít v počítači již nainstalovánu Javu 1.3 nebo novější.

Důvěrná informace: OmegaT Java Web Start neukládá žádné informace na server OmegaT. Aplikace běží pouze na vašem přístroji. Vaše dokumenty a překladové paměti zůstávají na vašem počítači a projekt OmegaT nemá žádný přístup k vaší práci nebo informacím.

Verze /trunk (alfa) odpovídá nejaktuálnějšímu stavu vývoje. Je doporučená pouze pro zkušené vývojáře a zkušební provoz. Tuto verzi můžete získat přes příkaz pro subverzi:

svn checkout svn://svn.code.sf.net/p/omegat/svn/trunk omegat-svn

Pravidelná sestavení jsou dostupná na Sourceforge.

Běžní uživatelé balíčky zdrojového kódu nepotřebují, jsou určeny pouze pro vývojáře. Naleznete je v adresářích odpovídajících kompilovaným verzím.
